// adversarial-nids-lab

Constraint Inflation — Interactive Demo

Most published adversarial NIDS evaluations run gradient attacks without constraining outputs to physically valid network traffic. This produces adversarial examples with impossible feature values — negative TTLs, sub-zero packet counts, rates outside [0,1] — that no real network can generate. Reported evasion rates are inflated by how easily the optimizer exploits this infeasible space.

This demo runs constrained PGD (physically valid) and unconstrained PGD (standard evaluation) side-by-side on real UNSW-NB15 network flows so you can see exactly what changes — and which features go impossible.
